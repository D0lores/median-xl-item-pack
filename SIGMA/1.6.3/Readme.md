## 装备库内容

## 使用方法
下载_LOD_SharedStashSave.sss，拷贝至魔电存档目录，配合MedianXLOfflineTools_0.5.6（离线工具）使用。

## FAQ
Q：如何循环使用？
A：把_LOD_SharedStashSave.sss文件的属性改成只读，这样从装备库拿出或者放入装备（会消失），保存之后都不会影响装备库，重新打开装备库就恢复原样。
